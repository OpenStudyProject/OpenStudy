# GitHubのアカウントを作成する

まだGitHubのアカウントを持っていない場合は作成してください。

# リポジトリをフォークする

OpenStudyのリポジトリを自分のアカウントにフォークします。

# Hugoのインストール

## MacOSの場合

HomeBrewから

```shell
brew install hugo
```

でインストールできます。

- `hugo`コマンド
- `hugo server`コマンド
- `http://localhost:1313/` にアクセスする
